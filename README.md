# Scalable E-commerce Platform in ArnoldC

A scalable e-commerce platform built with ArnoldC, the Arnold Schwarzenegger-based programming language.

## Project Structure

- `src/main.arnoldc`: Entry point of the application
- `src/product.arnoldc`: Product management functionality
- `src/cart.arnoldc`: Shopping cart operations
- `src/user.arnoldc`: User authentication and management

## Building and Running

1. Make sure you have ArnoldC installed
2. Run `make` to build the project
3. Run `./ecommerce` to start the application
